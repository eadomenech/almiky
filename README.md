# Almiky

## Python library for data hiding in images

### Packaging Project

1- Config virtualenv:

```bash
$ git clone https://gitlab.udg.co.cu/dprocessing/almiky.git
$ python3 -m venv env
$ source env/bin/activate
```

2- Build the package

```bash
$ pip install setuptools wheel
$ cd almiky/
$ python3 setup.py sdist bdist_wheel
```
